script

            
            var nome = window.prompt('Qual é seu nome?')
            window.alert('É um prazer te conhecer, ' + nome)

O prompt recebe um valor string

(number + number) para adição
(string + string) para concatenação

Para converter String -> Número

  • Number.parseInt(n)

    Número Inteiro

  • Number.parseFloat(n)

    Número Real (com virgulas e tals)

  • Number(n)

    JavaScript que decide se vai ser int ou float

Para converter Número -> String

  • String(n)
  • n.toString()
