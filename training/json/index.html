O que é JSON?

É um jeito de escrever dados, como se fosse um objeto do JavaScript. Ele é usado para trocar informações entre sistemas, principalmente entre API ↔ site.

Para que serve JSON?

Serve para enviar e receber dados. Exemplos:

  • Buscar CEP
  • Fazer login
  • Enviar formulários
  • Buscar lista de produtos
  • Carregar posts, comentários, vídeos
  • APIs (quase todas usam JSON)
